# 연결 리스트 <br/>

### 연결 리스트의 성질 <br/>

- k 번째 원소를 확인/변경하기 위해 **O(k)** 가 필요함
- 임의의 위치에 원소를 추가/제거는 **O(1)** 필요
- 원소들이 메모리 상에 연속해있지 않아 Cache hit rate 가 낮지만 할당이 다소 쉬움
